# SendBox - User Guide

Welcome to **SendBox**, a smart and automated tool designed to help you manage and process accounts efficiently. This guide will help you set up and use the software easily.

## üöÄ Getting Started

### 1. Installation
1.  Download the **SendBox** software package.
2.  Locate the file named `SendBox.exe`.
3.  Double-click `SendBox.exe` to launch the application.
    *   **Note:** On the first run, the software may take a few minutes to download necessary components (like the browser engine). Please be patient and let it finish.

### 2. Activation
To use SendBox, you need a valid license key.
1.  When you open the app, you will see your **PC ID** on the left sidebar.
2.  Click the **Copy** button next to the PC ID and send it to your software provider/admin to get your License Key.
3.  Once you have the key, click the **Change** or **Activate** button in the License section.
4.  Paste your key and click **Activate**.
5.  Once activated, you will see your license status and expiry date.

## üõ†Ô∏è How to Use

### Step 1: Add Numbers
1.  Go to the **Number Manager** tab in the main window.
2.  Paste your list of numbers into the text box.
3.  Click **+ Add to Queue**.
4.  Your numbers will appear in the list below.

### Step 2: Add Proxies (Recommended)
Using proxies helps prevent restrictions.
1.  Go to the **Proxy Manager** tab.
2.  Paste your proxies in the text box (Format: `IP:Port` or `IP:Port:User:Pass`).
3.  Click **+ Add Proxies**.
4.  You can also use the **Get proxies** button if your provider supports auto-fetching.

### Step 3: Configuration
On the left sidebar, you can adjust the settings:
*   **Threads:** Use the slider to set how many accounts to process at the same time.
    *   *Recommendation:* Start with **4-5 threads**. If your PC is powerful, you can increase it.
*   **Headless Mode:**
    *   **ON (Recommended):** The browser works in the background (hidden). This is faster.
    *   **OFF:** You will see the browser windows opening and working. Use this if you want to watch the process.

### Step 4: Start the Process
1.  Once you have added numbers and proxies, click the green **START PROCESS** button on the left.
2.  The status badge at the top will change to **RUNNING**.
3.  You can monitor progress in the **Live Logs** tab.

## üìä Viewing Results

You can see the results of the operation in two ways:

1.  **In the App:**
    *   Go to the **Results Data** tab to see a table of successful accounts (`Valid`).
    *   The **Stats Cards** at the top show a summary of Valid, Invalid, and other statuses.

2.  **Saved Files:**
    *   The software automatically saves results to a `Results` folder on your computer.
    *   Look for files like `Valid_Accounts.txt` in the software directory or the `Results` folder.
